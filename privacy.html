개인정보 처리방침

cue.(이하 "서비스")는 사용자의 개인정보를 소중히 여깁니다. 본 방침은 cue.가 수집하는 정보와 사용 방법을 설명합니다.

1. 수집하는 정보

- 이메일 주소 (회원가입 시)

- 사용자 이름 및 프로필 정보 (사진 포함)

- 업로드한 음악 포스트, 댓글, 좋아요, Drop 컨텐츠

- 음악 취향 데이터 (선호 장르, 바이브 설정)

- 상호작용 로그 (좋아요, 팔로우, 재생 활동)

- 푸시 알림 토큰 (알림 수신 동의 시)

- 기기 정보 및 앱 버전 (오류 진단 목적)

- ISRC 코드 (음악 트랙 식별자, 공유된 트랙에 포함)

2. 정보 사용 목적

- 서비스 제공 및 개인화 (음악 추천, 취향 기반 피드)

- 사용자 간 소통 기능 제공

- 푸시 알림 발송

- 서비스 품질 개선 및 오류 수정

수집된 정보는 제3자에게 판매되지 않습니다.

3. 제3자 서비스

cue.는 다음 서비스를 사용합니다:

- Supabase: 데이터베이스 및 인증 (supabase.com/privacy)

- Deezer: 음악 검색, 앨범아트, 미리듣기 (deezer.com/legal/personal-datas)

- YouTube Data API: 조회수 데이터 (policies.google.com/privacy)

- Billboard Hot 100: 공개 차트 데이터 (GitHub 공개 저장소 경유, billboard.com)

4. 데이터 저장 위치

모든 사용자 데이터는 일본 도쿄 (ap-northeast-1) Supabase 서버에 저장됩니다. 데이터는 암호화된 상태로 전송 및 저장됩니다.

일부 데이터 (최근 재생 목록, 캐시)는 사용자 기기에만 저장되며 서버로 전송되지 않습니다.

5. 데이터 보관

계정 삭제 시 모든 개인정보(게시물, 댓글, 좋아요, 팔로우, 인증 정보 포함)는 즉시 삭제됩니다.

6. 사용자 권리

사용자는 언제든지 본인의 데이터 열람, 수정, 삭제를 요청할 수 있습니다. 앱 내 설정 > 계정 삭제 메뉴를 통해 직접 삭제하거나, 아래 이메일로 요청할 수 있습니다.

7. 미성년자 보호

cue.는 만 14세 미만의 어린이를 대상으로 하지 않습니다. 14세 미만 사용자의 개인정보가 수집되었음을 인지한 경우 즉시 삭제 조치합니다.
